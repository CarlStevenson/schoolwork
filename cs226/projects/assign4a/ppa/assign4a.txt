Description:

  The main program drives the list class, which is an integer singly linked list. The list class has functions to display itself, find an element in itself, insert an item into itself, delete elements from itself, and return the number of elements in itself. The main program displays a menu and takes input in order to perform these functions in the class.

Valid input:

  For the main program, an integer x where 1<= x <=6.
  For the findElement function, an integer x.
  For the insertElement function, an integer x.
  For the deleteElement function, an integer x.
